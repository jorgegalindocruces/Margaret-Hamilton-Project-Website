En **Margaret Hamilton Project** no solo enseñamos programación a niñas y niños: también creemos firmemente en el **conocimiento compartido**. Por eso, hemos dado un paso más en nuestro compromiso con la divulgación: **hemos liberado el código fuente de nuestra página web**.

### ¿Por qué Open Source?

Margaret Hamilton, la científica que inspira nuestro proyecto, lideró el desarrollo del software que llevó al ser humano a la Luna. Su trabajo no solo fue innovador por su calidad técnica, sino también por su enfoque colaborativo y su compromiso con la documentación y el conocimiento compartido.

Siguiendo ese espíritu, creemos que:

- **El conocimiento debe ser accesible para todos**, no solo para quienes pueden pagarlo.
- **Compartir código es compartir oportunidades** para que otros proyectos educativos, ONGs o iniciativas sin ánimo de lucro puedan crear sus propias webs.
- **La transparencia genera confianza**, y queremos que cualquiera pueda ver exactamente cómo funciona nuestra web.

### ¿Qué puedes encontrar en el repositorio?

Hemos publicado todo el código en GitHub:

👉 **[github.com/jorgegalindocruces/margarethamilton](https://github.com/jorgegalindocruces/margarethamilton)**

El proyecto incluye:

- **Página web completa** desarrollada con Next.js 14 y React
- **Sistema de contenidos** basado en archivos MDX para blog, cursos y eventos
- **Formularios funcionales** conectados con Formspree
- **Diseño responsive** con Tailwind CSS
- **Optimización SEO** con sitemap automático y robots.txt
- **Integración con Google Analytics**
- **Despliegue automático** en GitHub Pages

### Desarrollada con ayuda de Inteligencia Artificial

Esta web ha sido desarrollada con la ayuda de **[Claude Code](https://claude.ai/claude-code)**, el asistente de programación de Anthropic. Creemos que las herramientas de IA pueden democratizar el desarrollo de software, permitiendo que proyectos sin ánimo de lucro como el nuestro puedan crear webs profesionales sin necesidad de grandes presupuestos.

Claude Code nos ha ayudado a:
- Estructurar el proyecto con las mejores prácticas
- Implementar funcionalidades complejas de forma eficiente
- Optimizar el SEO y el rendimiento
- Resolver problemas técnicos en tiempo real

Esto también forma parte de nuestra filosofía de **transparencia**: no solo compartimos el código, sino también cómo lo hemos creado.

### Tecnologías utilizadas

Para los más curiosos, estas son las tecnologías que hemos usado:

| Tecnología | Uso |
|------------|-----|
| **Next.js 14** | Framework de React para sitios estáticos |
| **TypeScript** | Tipado estático para código más robusto |
| **Tailwind CSS** | Estilos y diseño responsive |
| **MDX** | Contenido dinámico con Markdown + React |
| **Formspree** | Gestión de formularios sin backend |
| **GitHub Pages** | Hosting |
| **GitHub Actions** | Despliegue automático |
| **Claude Code** | Asistente de IA para desarrollo |

### Cómo usar nuestro código

Si quieres crear una web similar para tu proyecto, puedes:

1. **Clonar el repositorio**
   ```bash
   git clone https://github.com/jorgegalindocruces/margarethamilton.git
   ```

2. **Instalar dependencias**
   ```bash
   npm install
   ```

3. **Ejecutar en local**
   ```bash
   npm run dev
   ```

4. **Personalizar** el contenido, los colores y las imágenes según tu proyecto.

### Parte de nuestro compromiso con la divulgación

Esta iniciativa forma parte de nuestro compromiso más amplio con la **divulgación y la educación abierta**:

- 📚 **Cursos** en YouTube para aprender Scratch
- 📖 **Materiales educativos** disponibles para otros educadores
- 💻 **Código abierto** de nuestra web
- 📊 **Transparencia financiera** con informes públicos de gastos e ingresos

Creemos que el conocimiento crece cuando se comparte. Si cada proyecto educativo tuviera que empezar de cero, avanzaríamos mucho más lento. Al compartir lo que hemos construido, esperamos que otros proyectos puedan ir más lejos y más rápido.
